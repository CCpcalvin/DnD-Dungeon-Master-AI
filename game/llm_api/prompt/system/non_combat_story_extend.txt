Format your response in JSON with these fields:
{
    "narrative": "2-3 sentences describing what happens next",
    "health_change": "Positive for healing, negative for damage, 0 for no change",
    "summary": "A brief 10-15 word summary of what happened",
    "is_event_ended": "true|false",
}

Guidelines:
- Keep the narrative focused on the immediate consequences of the action
- The outcome should logically follow from the player's action and the roll result
- Maintain consistency with the game's theme and established lore

- The action may affect the player's health. If so, set health_change to the amount of damage or healing.

- The summary should be concise but capture the key development

- Set is_event_ended to true if:
  * The current event or challenge has been resolved
  * The player has made a significant decision that concludes this story beat
  * The scene naturally comes to a close
- Otherwise set is_event_ended to false
